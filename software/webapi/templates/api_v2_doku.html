TheOpenVent API v2 documentation

Description of the parameters

General parameters & groups

Parameter Unit Data type Comment
device_id - unsigned integer unique identifier of the ventilator
processed - group human readable parameters, which are calculated from raw values
trigger_settings - group Settings of the ventilator
raw - group raw values coming directly from the sensors
time seconds unsigned integer the Unix timestamp of the data

Processed parameters

Parameter Unit Data type Comment
volumePerMovement mL unsigned integer total inspired volume per breath cycle
frequency 1/minute unsigned integer total number of breathes per minute
MVe mL integer total inspired air over one minute
ventilationMode - string type of the control e.g: Assist Control/ Pressure Control/ Volume Control/ Flowrate Control
ExpiredCO2 % float concentration of expired CO2
ExpiredO2 % float concentration of expired O2
flowrate mL/min unsigned integer flowrate
pressure cmH20 integer air pressure

Trigger settings

Parameter Unit Data type Comment
FiO2 % float O2 concentration of inspired air
humidity % float Air humidity of inspired air
pressure_max cmH2O unsigned integer Maximum allowed peak pressure
RR 1 / minute unsigned integer Breathes per Minute
VT mL unsigned integer Tidal Volume
PEEP cmH2O float Positive End Expiratory Pressure
IE - float Inspiration to Expiration rate: 1.4 = 1 to 4

Raw parameters

Parameter Unit Data type Comment
pressure1 cmH2O unsigned integer Raw data of pressure sensor 1
pressure2 cmH2O unsigned integer Raw data of pressure sensor 2
temperature1 Celsius float Raw data of temperature sensor 1
temperature2 Celsius float Raw data of temperature sensor 2
angleSensor degree float Raw data of angle sensor
current A float Raw data of current sensor
motorRPM 1 / minute integer Motor rounds per minute
